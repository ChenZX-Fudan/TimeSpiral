# ⏳ 时间螺旋 · TimeSpiral

> 你是一只名叫 **12号** 的猫武士。时间之河正在被暗影侵蚀。探索、对话、解谜、战斗——守护时间螺旋的平衡。

🎮 在线版：`https://chenzx-fudan.github.io/TimeSpiral/`

---

## 📖 这是什么游戏？

一个**点触式探险游戏**。你会在浏览器里看到一张张手绘风格的场景地图，上面散布着可点击的图标。点击它们来探索世界、和守护者对话、解开谜题、击败暗影。全部操作只需**点一下鼠标**。

类比的话，它有点像密室逃脱 + 视觉小说的结合——没有复杂的操作，一切靠点击推进。

## 🎮 第一次玩，会发生什么？

### 1. 标题画面

看到封面和两个按钮：

| 按钮 | 效果 |
|------|------|
| ✨ 开始冒险 | 从零开始，只有序章解锁 |
| 📖 继续冒险 | 读取上次自动存档，接着玩 |

### 2. 章节选择

第一次只能点 **序章：时间的织者**。完成一章后自动解锁下一章，8 章全通后解锁番外。

```
序章 → 第1章 → 第2章 → ... → 第8章 → 🔒番外
```

### 3. 故事阅读

每章开头有一段故事文本（来自原创小说）。你可以读完它，也可以点「跳过」直接进入游戏。

### 4. 进入游戏 — 这是核心部分 🔥

你会看到这样的画面：


## 🎒 顶栏按钮

游戏顶部有一排圆形按钮：

| 按钮 | 功能 | 什么时候用 |
|------|------|-----------|
| 🎒 | 背包 | 查看已收集的物品。在特定场景使用物品可以推进剧情（比如在熔炉使用时间水晶） |
| ⚡ | 时间感知 | 消耗 20 能量，在当前场景显示隐藏提示（持续 2.5 秒） |
| 🗺️ | 世界地图 | 查看完整地图，标注当前位置和已探索区域 |
| 📖 | 章节选择 | 回到章节选择页面（自动存档） |

## ⚔️ 战斗怎么打？

1. 点击 👾 敌人图标进入战斗
2. 看到敌人的血条和三个技能按钮
3. **点击敌人** = 普通攻击（约 8 点伤害，免费）
4. 点击技能按钮消耗能量释放强力攻击
5. 敌人 HP 降到 0 = 胜利
6. 如果被打败，重新来过即可

```
        👹 暗影战士·卡奥斯
   ═══════════════════════  ← 血条
   "点击它来攻击！"

   [⚔️ 攻击 20]  [✨ 时间直觉 15⚡]  [💫 螺旋共鸣 30⚡]
```

## 💡 新手提示

1. **先和所有 NPC 对话** — 序章的目标就是和 3 位守护者交谈，别跳过对话
2. **能量省着用** — 隐藏提示很诱人，但战斗技能也需要能量。多找 💧 时间之泉免费回复
3. **背包物品在特定场景才能用** — 拿到时间水晶后，去裂缝区域的熔炉才能锻造钥匙
4. **自动存档** — 不用担心丢失进度，随时可以关掉浏览器
5. **卡住了？** — 点 ⚡ 时间感知看提示，或者打开 🗺️ 地图确认自己在哪
6. **阅读故事** — 别急着跳过！故事文本本身就是一个完整的猫武士小说

## 📁 技术栈

- 纯静态 HTML + CSS + JavaScript
- 单文件架构（`index.html` + `game.css` + `main.js`）
- 无框架、无构建步骤、无外部依赖
- 存档：`localStorage`

```
├── index.html
```
