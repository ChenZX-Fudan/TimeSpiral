# ⏳ 时间螺旋 · TimeSpiral

点触探险游戏。你扮演 **12号守护者**，在时间螺旋中探索、解谜、战斗，阻止暗影力量的入侵。

## 🚀 运行

```bash
cd e:\Claude\TimeSpiral
python -m http.server 8081
```

浏览器打开 `http://localhost:8081`

---

## 🎮 怎么玩

### 游戏流程

```
标题画面 → 选择章节 → 阅读故事（可跳过）→ 进入游戏
                                              ↓
                           完成目标 → 章节通关 → 解锁下一章
                                              ↓
                        全部8章通关 → 解锁番外
```

### 操作

所有操作都是**点击**：

- **标题画面** — 开始冒险 / 继续冒险
- **章节选择** — 点击已解锁的章节进入。完成一章后解锁下一章，8章全通后解锁番外
- **故事阅读** — 完整显示本章故事文本，可直接阅读，也可点击「跳过」或底部「进入冒险」
- **游戏中** — 点击地图上的图标探索、对话、解谜、战斗

### 顶部按钮

| 按钮 | 功能 |
|------|------|
| 🎒 | 打开背包，使用物品 |
| ⚡ | 消耗20能量，显示隐藏提示 |
| 📖 | 返回章节选择 |

### 章节列表

| 章节 | 目标 |
|------|------|
| 序章：时间的织者 | 与守护者们对话，了解时间之源 |
| 第一章：时光的回响 | 前往裂缝区域，击败暗影斥候 |
| 第二章：暗影初现 | 收集水晶，锻造钥匙，打开封印 |
| 第三章：风暴前夕 | 深入暗影之地，击败暗影战士 |
| 第四章：决裂时刻 | 穿越裂缝，寻找14号 |
| 第五章：拯救之旅 | 收集能量，准备决战 |
| 第六章：暗影侵袭 | 击败所有暗影生物 |
| 第七章：守护者的牺牲 | 面对最终挑战 |
| 第八章：时间的重生 | 封印暗影核心 |
| 🔒 番外1 | 完成全部8章后解锁 |

### 能量系统

- 100 点时间能量
- ⚡ 消耗 20 点显示隐藏线索
- 战斗中技能也消耗能量
- 时间之泉和能量节点可免费回复

### 谜题

- **记忆配对** — 翻开符文方块，找出相同配对
- **时间序列** — 按 月→星→日→旋→光 顺序点击

### 战斗

点击怪物 + 技能按钮，敌人 HP 归零即胜。失败可重试。

### 存档

自动存档。也可在章节选择页手动重置进度。
